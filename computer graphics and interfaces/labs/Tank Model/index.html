Camera:

  • 1: Set view 1.
  • 2: Set view 2.
  • 3: Set view 3.

  • - Zoom out.

Movement:

  • ArrowUp: Move tank forward.
  • ArrowDown: Move tank backward.
  • A: Move top body left.
  • D: Move top body right.

Weapon:

  • W: Increase gun movement.
  • S: Decrease gun movement.
  • Space: Shoot.

View:

  • Caps-lock + W: Switch to Lines mode.
  • Caps-lock + S: Switch to Object mode.
